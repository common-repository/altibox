Altibox is a Lightbox Image Viewer manageable through CSS selectors.

== Description ==

Altibox is a clean and light **Image Viewer** for WordPress controlled by CSS selectors. It displays your images without opening a new browser window or tab. Your visitors remain in a same page for a smooth user experience.

* Lightweight Image Viewer is only **3.9kB** (Javascript and CSS files).
* Use CSS selectors to manage when to open Image Viewer.

Available languages :

* English

For support, please visit [Altibox plugin](http://www.alticreation.com/en/altibox/ "altibox image viewer plugin for Wordpress by alticreation")

== Installation ==

1. Upload `altibox` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Support =
